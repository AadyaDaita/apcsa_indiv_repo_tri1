Benefit(s) of backend implementation?

Implementing APIs in backend allows us to control specific logistics about how the API is run, when otherwise, it would not be possible. For example, if there is a limit on how many times an API could be run, the backend code will make sure to not run the API past that limit. In my case, you can only run the API 500 times a month, so the backend code makes sure that it meets the limit.
